OpenX RTMP Video Ad Plugin for Flowplayer - Beta Version 0.4.6

Welcome to the beta version (0.4.6) of the RTMP video advertising plugin for OpenX and Flowplayer.

This package contains the binary, source and build files needed to extend the plugin. If you are looking for high level instructions on how to build this release - read this. Several examples are also provided to help illustrate the various configurations for the plugin.

Further documentation can be found:

  • On the main plugin page at www.flowplayer.org
  • On the detailed plugin documentation pages at www.bouncingminds.com
